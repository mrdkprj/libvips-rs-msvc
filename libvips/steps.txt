1. Extract DLL
From the ZIP:
bin/libvips-42.dll

2. Generate .def from DLL
Use pexports or gendef:

pexports libvips-42.dll > libvips-42.def
gendef libvips-42.dll

Confirm: libvips-42.def starts with cat libvips-42.def

3. Generate MSVC import .lib
Using lib.exe from Visual Studio:
x64 Native Tools Command Prompt for VS

lib /def:libvips-42.def /machine:x64 /out:libvips-42.lib

4.link
